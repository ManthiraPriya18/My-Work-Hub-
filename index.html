    KEY-VAULT

    Key Vault is a secure information storage application designed to protect user data with encryption. It ensures that only authenticated users can access their stored information through a secure login system. Built using Flask, Python, SQLAlchemy, and SCSS, the app offers a seamless and user-friendly interface while maintaining high security. By integrating encryption techniques, Key Vault guarantees that sensitive data remains protected from unauthorized access, making it a safe and reliable solution for personal and confidential information storage.

    KEY-VALUT 2.0

    Key Vault 2.0 is an enhanced and more secure version of Key Vault, designed to provide highly protected data storage using a two-step encryption mechanism. This approach encrypts the already encrypted data before storing it in Supabase, a PostgreSQL-powered open-source backend known for its real-time database management, built-in authentication, and secure storage solutions. This ensures maximum security and protection against unauthorized access. This version maintains user authentication and secure login mechanisms while improving data integrity and confidentiality. It is built using React + Vite for a faster, optimized front-end experience and Supabase as the database solution. With Key Vault 2.0, users get a seamless, fast, and highly secure way to store their sensitive data, making it an ideal privacy-focused solution.
